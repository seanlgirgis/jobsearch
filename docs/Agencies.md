## How to apply on recruiter sites

Do it this way:

1. Open the recruiter job page.
2. Save the full job description into `intake.md`.
3. Run A1 with me first.
4. If APPLY or STRONG APPLY, generate the tailored resume.
5. Apply on the recruiter site with the tailored resume.
6. Immediately email the recruiter listed on the posting, if available.

The email matters because recruiter portals can be black holes.
