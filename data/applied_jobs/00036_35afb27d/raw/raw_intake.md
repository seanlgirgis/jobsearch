Job#: 3022685 Job Description: Job Title: Performance Lead Engineer Contract Duration: 9 Months Location: Plano, TX 
•	Individual will provide performance testing capabilities for complex applications. He/she will work with stakeholders on a daily basis to design test cases, create performance test scripts and execute performance scenarios.
•	Works under the direction and with a Performance Lead
•	Writes technical performance test plans
•	Requires specialized Python programming skills
•	Develops data driven load test scripts using Citrix Protocol
•	Uses Load Runner (or JMeter) to design and execute load and performance tests
•	Lead the data creation activities for the required load tests
•	Designs and executes different types of load tests (load, failover, aging etc)
•	Works closely with application development, architecture, infrastructure and production support groups for defect triage and production planning.
•	Reports findings according to a defined process, such as sending an email.
•	Participate in project/team meetings and actively contribute.
•	About 5-10 years of previous hands on experience as developer and/or performance architect
•	An understanding of web protocols including HTTP, HTTP, Citrix etc.
•	Any experience in any load management applications like LoanIQ (Finastra). If someone has development experience, that will be better.
•	Must be well versed in using VUGen 9.5 or greater for advanced test script development beyond the built-in record and playback capabilities.
•	Experience in Programming languages such as C, Java, SQL query development, Scripting, etc. Also requires specialized Python programming skills
•	Ability to use different monitoring/profiling tools (Dynatrace, CA Wily Introscope, AWR reports etc.) to monitor, gather, and report application performance on Windows or Linux machine
•	Knowledge of Splunk to be able to do better load test analysis
•	Has an understanding of the distributed load test environments, and the application architecture
•	Knowledge of performance bottlenecks and end-to-end performance web performance measures
•	Strong analytical skills with a natural ability to research and resolve complex systems, applications and user issues
•	Excellent verbal and written communication skills
•	Ability to troubleshoot issues in the load and performance test environment LoadRunner; Ability to use different monitoring/profiling tools (Dynatrace, CA Wily Introscope, AppDynamics, AWR reports etc.) to monitor, gather, and report application performance on Windows or Linux machines ;
•	Has an understanding of the distributed load test environments, and the application architecture
